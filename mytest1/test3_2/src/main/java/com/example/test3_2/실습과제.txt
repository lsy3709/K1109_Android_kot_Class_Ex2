예시 클래스
1. User , ID: 문자열, PW : 문자열, email : 문자열, phone : 문자열
2. Login 처리 클래스 : ID, PW 일치 하면 , 콘솔에 출려하는 예제
3. Register 처리 클래스 : Scanner로 입력을 받아서 , 로그인이 되면 콘솔 출력.
4. 실행 클래스
---------------------------------------
뷰에서 : 회원 가입 정보를 입력 받는 양식 콘솔에서 입력 받고
입력 받은 정보를 기준으로 로그인 처리 후, 결과를 콘솔에 출력하는 예제.

입력 받은 정보를 User 라는 클래스 객체에 인자로 사용하고,
더미 객체 임시 객체 ( 등록된 사용자 예시 ex) admin / 1234
일치하면 로그인 되었다고 콘솔에 해당 유저 출력.
